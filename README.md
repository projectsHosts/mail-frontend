# Bulk Email Sender (Frontend)

A clean and modern web-based UI to create and manage bulk email campaigns.  
This project is built with Angular and designed for educational institutions, coaching centers, training institutes and student communities.

| Pagination & Sorting | Easy navigation for large lists |
